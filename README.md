# ФинПоток - Единая платёжная сеть

Современный лендинг для платёжной платформы ФинПоток.

`tags: ["landing", "dark", "fintech", "glass"]`

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Превью

```bash
npm run preview
```
